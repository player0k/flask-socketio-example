# Flask SocketIO Example #

## Dependence ##
* Python 2 or 3
* Flask >=0.10.1
* Flask_SocketIO >=2.2
* eventlet

## Run ##
```
python ./scripts/main.py
```
Open `http://localhost:5000`.
